#CMake SFML Template 

SFML template project to start developing any app with the C++ framework.
Tested on windows as for now.

### Windows users
You need to copy the dll's to the build folder
* openal32.dll
* sfml-audio-2.dll
* sfml-audio-d-2.dll
* sfml-graphics-2.dll
* sfml-graphics-d-2.dll
* sfml-system-2.dll
* sfml-system-d-2.dll
* sfml-window-2.dll
* sfml-window-d-2.dll
